**********PROJE ÖZETİ**********

->Bu proje bizden istenen 5 adımı 5 farklı html sayfasından yapmamız istenmektedir, 
=> Bu 5 adım sırasıyla bizim yaptığımız şekilde söyleyecek olursak, 
 -) 1. adım için verilen url içerisinde kelimelerin frekansını bulmak, 
 -) 2. adım için frekansı en yüksek olan 5 kelime anahtar kelime yapmak, 
 -) 3. adım için Cosine Similirsity algoritmasıyla girilen 2 urlin benzerlik karşılaştırılması yapılması
 -) 4. adım için girilen url veya url kümesinin içerisindeki linkleri bulmak ve tüm url'leri main url ile 3. adımdaki gibi karşılaştırmak
 -) 5. adım için girilen url veya url kümesinin içerisindeki linkleri bulmak ve tüm urlleri main url ile semantik analiz yaparak karşılaştırmak

**********PROJEYİ İNDİRMEK**********

-> kaynakKod.zip kodunun içerisinde mevcuttur.
-> 180201011-180201159.txt ise node modülü hariç tüm html ve nodejs kodu mevcuttur.
NOT : >.zip dosyasını açmak için 
>Windows'ta çeşitli programlar kullanabilirsiniz. 
>Linux'ta ise eğer .zip dosyasında şifre yoksa unzip dosyaadı.zip / eğer .zip dosyasında şifre var ise unzip -P şifre dosyaadı.zip ile açabilirsiniz.

**********PROJEYİ ÇALIŞTIRMAK VE TEST ETMEK**********

-> kaynakKod.zip içeriğini çıkarınız ve önerilen olarak Visual Studio code içerisinde açınız sorunsuz açılacaktır.
-> terminali açınız
-> ilk olarak npm i ile node modülü indirin veya updateleyin
-> node main.js yazarak run edebilirsiniz, 
-> run ettiğinizden sonra localhost:3000 port'unda proje çalışacaktır
